# Fiche de style

## Couleurs

### Principales
- **Primary** `#FF6A00`
- **Primary Dark** `#C65300`
- **Secondary** `#FF8A3D`

### Secondaires / neutres
- **Gris foncé** `#222222`
- **Beige clair** `#F5F1EB`

Ces teintes sont appliquées via les classes existantes (par ex. `.section--dark` et les sections alternées) pour assurer une cohérence visuelle et un contraste accessible.
