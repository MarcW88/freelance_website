# Marc Williame - Consultant SEO orienté Data & IA

## 🎯 MVP Complet - Site Freelance SEO

Site web professionnel pour Marc Williame, consultant SEO spécialisé en data et intelligence artificielle.

### ✨ Caractéristiques

- **Design System moderne** inspiré de Linear.app, Vercel et Refraction.dev
- **Palette cohérente** : Bleu nuit (#0E1117) + Turquoise (#18D4B7) + Blanc
- **Typography** : Space Grotesk (titres) + Inter (corps) + IBM Plex Sans (CTA)
- **Animations fluides** avec AOS et micro-interactions
- **Responsive design** optimisé mobile-first
- **SEO-ready** avec Schema.org et meta optimisées

### 🏗️ Structure

1. **Hero Section** - Dégradé sombre avec grille animée
2. **Intro** - Différenciateurs en 3 colonnes
3. **Services SEO** - 4 cartes avec hover effects
4. **Data & IA** - Section signature sur fond sombre
5. **Méthodologie** - Timeline horizontale
6. **À propos** - Layout photo + contenu
7. **Contact** - Formulaire avec validation
8. **Footer** - Minimaliste et élégant

### 🎨 Design System

#### Couleurs
```css
--color-dark: #0E1117        /* Fond sombre principal */
--color-white: #FFFFFF       /* Fond clair */
--color-gray-light: #F9FAFB  /* Sections alternées */
--color-accent: #18D4B7      /* Accent turquoise */
--color-text: #111827        /* Texte principal */
```

#### Typography
- **Titres** : Space Grotesk Bold (moderne, "builder")
- **Corps** : Inter Regular (lisible, sobre)
- **CTA/Labels** : IBM Plex Sans Medium (technique)

#### Spacing
- Sections : 96px verticaux (--space-4xl)
- Éléments : Système cohérent de 0.5rem à 6rem

### 🔄 Intégration Avada

Le code est prêt pour intégration dans Avada WordPress :

1. **CSS** → Avada > Options > Custom CSS
2. **HTML** → Sections Avada Builder
3. **JavaScript** → Footer scripts
4. **Fonts** → Avada > Typography

#### Sections Avada
- Utiliser **Container** avec classes CSS existantes
- **Colonnes** pour les grilles (services, intro, etc.)
- **Éléments texte** avec classes personnalisées
- **Boutons** avec classes .btn
